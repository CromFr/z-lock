- How to install.

Unpack z-lock.zip, and execute 'z-lock.exe'.


- How to operate.

-- Movement     Arrow / Num / [WASD]   / Joystick

-- Shot         [Z][/]         / Trigger 1

Hold a shot key to open automatic fire.
It serves also as adsorption in "Concept-mode."

-- Adsorption   [X][BackSlash] / Trigger 2 (Normal-mode only)

The "lock on marker" released out of the enemy is made to stick to a self-opportunity.
It is not used in "Concept-mode."

-- Pause        [P]            / Trigger 3


- How to play.

There are the 4 modes.

-- Normal-mode

It is the usual mode. A shot can be shot even if lock on is not carried out.

-- Concept-mode

If not locked by the enemy, it is the mode with which a shot cannot be shot.

-- Original-mode

It is the mode which became the prototype of this game.

-- Hidden-mode

It is the mode in which enemy shells disappear gradually.

-- Score attack

It is the mode which competes for the ability to be earned in 3 minutes.

-- Time attack

It is the mode which competes for time until it takes 1 million points.
